## How to Install

1. **Install Ollama**
   ```sh
   curl -fsSL https://ollama.ai/install.sh | sh
   ```

2. **Download DeepSeek-R1 from the website**
   ```sh
   ollama pull deepseek-r1
   ```

3. **Check the downloaded LLMs**
   ```sh
   ollama list
   ```

4. **Start the LLM server**
   ```sh
   ollama serve
   ```
